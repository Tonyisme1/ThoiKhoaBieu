# Hướng dẫn sử dụng Thời khóa biểu

## Giới thiệu
Đây là một dự án thời khóa biểu đơn giản được xây dựng bằng HTML, CSS và JavaScript.

## Cách sử dụng
1. Mở file `index.html` trong trình duyệt web của bạn.
2. Thời khóa biểu sẽ được hiển thị trên trang.

## Dữ liệu
Dữ liệu thời khóa biểu được lưu trữ trong file `Lich_Hoc.json`. Bạn có thể chỉnh sửa file này để thay đổi nội dung thời khóa biểu.
